Why Lightning CSS?

Lightning CSS can take modern CSS features like nesting and transform them into browser-friendly CSS while also optimizing the output.

What This Page Shows

A simple layout with a header, two cards, and responsive styling. The original CSS uses nesting, and Lightning CSS can compile it into regular CSS.
